## Flight Delay Prediction & Aviation Analytics Platform

This project is an end-to-end big-data analytics and machine-learning platform developed to analyze and predict flight delays across the United States. It integrates approximately 21.2 GB of historical flight, airfare, airport, and NOAA weather data covering the period from 2009 to 2018.

The data was cleaned, transformed, and joined using Python, PySpark, and Spark SQL to create a scalable analytical pipeline. Exploratory analysis was conducted to identify delay patterns associated with airports, routes, airlines, seasonal trends, departure times, and weather conditions.

Machine-learning models were trained and evaluated to estimate the likelihood of a flight delay using operational and weather-related features. The final results were incorporated into an interactive Flask web application that allows users to explore airport statistics, historical delay trends, route-level insights, weather effects, and flight-delay predictions.

### Key Features

* Processed approximately 21.2 GB of multi-source aviation and weather data
* Developed scalable ETL and feature-engineering pipelines using PySpark
* Integrated historical flight records with NOAA weather observations
* Analyzed airport, route, seasonal, temporal, and weather-related delay patterns
* Trained and evaluated machine-learning models for flight-delay prediction
* Created interactive visualizations and analytical dashboards
* Deployed project insights and predictions through a Flask web application
* Designed the system to translate large-scale transportation data into actionable operational insights

### Technologies

Python, PySpark, Spark SQL, Pandas, NumPy, Scikit-learn, Flask, HTML, CSS, JavaScript, Matplotlib and NOAA weather data.
